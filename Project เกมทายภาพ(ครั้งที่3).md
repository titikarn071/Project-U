## การประชุมประจําวันที่ 2568/12/09

| ชื่อ                     | รหัส           | หน้าที่       |
| ---------------------- |:-------------:| ---------: |
| นายคณิศร เนตรสุวรรณ      | 68030030      | (ประธาน) |
| นางสาวฐิติกาญจน์ รัตนะเอี่ยม | 68030071      | (เลขานุการ) |
| นางสาวนภัสสร คำปัน       | 68030129      | - |


วันนี้ได้เขียนโครงร่างเกมส์ทายภาพ และรวบรวมความคิดเห็นของสมาชิกในทีมเพื่อนำความคิดเห็นของทุกๆคนมาปรับใช้ในการทำ Project. 
โดยที่ความคิดเห็นของสมาชิกในทีมทุกคน คิดเห็นว่าเราจะสร้างเกมทายภาพโดยที่ เราจะมีภาพให้ 2 ภาพ โดยทั้ง 2 ภาพจะประกอบไปด้วย ภาพที่ผิด 1 ภาพและภาพที่ถูก 1 ภาพ ให้ผู้เล่นเลือกภาพให้ถูกต้อง เพื่อให้สามารถไปในด้านถัดไปได้  
โดยเราจะแบ่งเป็นหมวดต่างๆ โดยที่แต่ละหมวดจะสอดแทรกความรู้ในรายวิชาต่างๆให้กับผู้เล่น
